Welcome to Ecommerce Website API

Welcome to the Ecommerce Website API. This API empowers developers to interact with a variety of features, from managing products and categories to handling user carts and orders.

Tech Stack

Ecommerce Website API is built using Express.js and PostgreSQL with Sequelize.js for the ORM. It also integrates Stripe for payment processing.

The project is deployed on a VPS and uses GitHub Actions for continuous integration and deployment.
